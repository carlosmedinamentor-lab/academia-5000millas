# Academia 5.000 Millas v2.0

⛵ + 💎 = 🦁

Campus educativo de la **Base de Lanzamiento (BDL)**.

## Método BDL

| Fase | Símbolo | Enfoque |
|------|---------|---------|
| **Barco** | ⛵ | Autoconocimiento — El viaje hacia tu interior |
| **Diamante** | 💎 | Transformación — El carbón bajo presión se convierte en diamante |
| **León** | 🦁 | Liderazgo — Tu verdadera identidad restaurada |

## Estructura

```
├── index.html          Landing principal del campus
├── catalogo.html       Catálogo con filtros por categoría BDL
├── curso.html          Página individual dinámica (?id=)
├── desafios.html       Desafíos de transformación
├── css/academia.css     Estilos obsidiana + oro sagrado
├── js/cursos-data.js   Base de datos de cursos
├── js/componentes.js   Header, footer, cards, modales
├── vercel.json         Deploy config + headers de seguridad
└── package.json
```
