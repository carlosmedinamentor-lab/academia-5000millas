# Academia 5.000 Millas v1.0

Campus educativo de la **Base de Lanzamiento (BDL)**.

## Descripción

Academia 5.000 Millas es una plataforma educativa diseñada para facilitar el aprendizaje y la formación dentro del ecosistema BDL. El nombre hace referencia al compromiso de recorrer las "5.000 millas" necesarias para alcanzar la excelencia en el conocimiento.

## Estructura del proyecto

```
academia-5000millas/
├── index.html          # Página principal del campus
├── css/
│   └── styles.css      # Estilos del campus
├── js/
│   └── app.js          # Lógica principal de la aplicación
├── assets/
│   └── images/         # Recursos gráficos
├── pages/
│   ├── cursos.html     # Catálogo de cursos
│   ├── perfil.html     # Perfil del estudiante
│   └── about.html      # Acerca de la academia
└── README.md
```

## Características

- Catálogo de cursos organizado por categorías
- Perfil de estudiante con seguimiento de progreso
- Diseño responsive y accesible
- Interfaz intuitiva y moderna

## Cómo empezar

1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. Explora los cursos disponibles

## Tecnologías

- HTML5
